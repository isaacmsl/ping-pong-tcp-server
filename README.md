# Ping Pong TCP Server

Just a TCPServer can play ping-pong with a TCPClient. The code is an exercise presented on _Computer Networking: A Top-Down Approach_ by Kurose and Ross.

## How to execute

1. `$ javac TCPServer`
1. `$ java TCPServer`
1. ... Similar for `TCPClient`
1. Send "ping"
1. Pong!
